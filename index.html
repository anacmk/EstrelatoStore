Estruturação de Telas para o Figma (Documentação do Grupo)

1. Frame Principal (Desktop)

Dimensões recomendadas: 1440px x 900px ou 1280px x 800px para protótipo de computador.

Grid de Layout: Grid de 12 colunas, margem lateral de 80px e gutter (espaçamento entre colunas) de 24px.

Fontes: Use Plus Jakarta Sans ou Inter para um visual limpo e super-legível.

2. Organização por Componentes

Crie componentes reutilizáveis no figma para:

  • Card de Status: Com variante para "Alerta" e "Vencido"
  • Menu Lateral (Sidebar): Usando auto-layout
  • Status Badge: Verde (Seguro), Amarelo (Atenção), Vermelho (Vencido)

3. Variáveis e Estilos Locais

Configure a biblioteca de cores com:

#0f172a (Azul Escuro / Navy)

#eab308 (Amarelo / Dourado Estrelato)

#f8fafc (Cinza Claro / Fundo)
